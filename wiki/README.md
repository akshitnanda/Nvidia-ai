# Wiki Source Pack

This folder is a starter pack for the GitHub Wiki.

## Suggested Flow

1. Open the GitHub wiki for the repository.
2. Create the first page named `Home`.
3. Copy the contents of [Home.md](Home.md) into that page.
4. Create the remaining pages with the same titles as the files in this folder.
5. Copy [_Sidebar.md](_Sidebar.md) into the wiki sidebar once GitHub exposes it.

## Pages Included

- `Home.md`
- `Quickstart.md`
- `Demo-Workflow.md`
- `Architecture-Notes.md`
- `Release-Process.md`
- `_Sidebar.md`

These pages are intentionally short and practical so the wiki complements the README instead of duplicating it.
